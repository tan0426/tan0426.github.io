# 32. IS INITIAL, IS BOUND, IS ASSIGNED

## IS BOUND
해당 변수가 유효한 변수인지 확인. 해당 데이터 변수는 스택을 가지고 있지만, 스택에 의해 데이터가 지워지더라도 유효하지 않을 수 있다.
- 

## IS INITIAL
피 연산자가 initial인지 확인. 피 연산자가 타입에 맞으면, true.

## IS ASSIGNED
필드심볼에 의해 메모리에 assign 되었는지 확인. 값이 들어가 있으면 true.
